# PlayStation Shop Management System

A simple and practical web application for managing a PlayStation shop, built with **HTML, CSS, and JavaScript** using **Vibe Coding** and the help of **free ChatGPT**.

## Overview

This project is designed to help manage the daily workflow of a PlayStation gaming shop.  
It provides an easy way to control devices, track gaming sessions, manage drinks inventory, and monitor daily income — all from a clean browser-based interface.

## Main Features

- Manage PlayStation devices
- Start **Normal** or **Multiplayer** sessions
- Pause and resume sessions
- Cancel sessions started by mistake
- Stop sessions and calculate the total bill
- Continue the same customer with a new session
- Set session duration:
  - 30 minutes
  - 1 hour
  - 2 hours
  - Open session
- Get notified when a session duration ends
- Add drinks to a specific device
- Choose drink quantity
- Track drinks stock and availability
- Hide unavailable drinks when out of stock
- Edit and manage devices from a separate settings page
- Edit prices for PS4 and PS5 from one place
- Add, edit, and delete drinks
- Reorder devices using drag and drop
- Save all data in browser local storage
- Display daily total and session history

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **LocalStorage**

## Development Style

This project was built using **Vibe Coding**, with the help of **ChatGPT Free** for planning, structuring, debugging, improving UI, and adding features.

## How It Works

The project runs completely in the browser without any backend.  
All saved data such as devices, drinks, sessions, totals, and settings are stored using the browser's `localStorage`.

## Goal of the Project

The main goal of this project is to create a useful real-life management system for a PlayStation shop while improving frontend development skills through hands-on practice.
